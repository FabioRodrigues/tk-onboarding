# Travelperk Onboarding API

This API aims to be a recipe-ingredients CRUD

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install this API.

```bash
pip3 install -r requirements.txt
```

## Usage
### Testing:

```bash
make tests
```
### Creating migrations:

```bash
make create-migrations
```
### Migrating:

```bash
make migrate
```
### Running:

```bash
make run
```
